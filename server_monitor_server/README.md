# Server Monitor

Маленький HTTP-сервер, який повертає завантаження процесора,
використання оперативної пам'яті та диска у форматі JSON.

## Запуск

```bash
python3 main.py
```

Сервер за замовчуванням слухає `0.0.0.0:8000`.

```bash
curl http://localhost:8000/stats
```

Адресу і порт можна змінити змінними середовища:

```bash
HOST=127.0.0.1 PORT=9000 python3 main.py
```
